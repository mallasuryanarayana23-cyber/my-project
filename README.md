# SkillMap

SkillMap is a production-ready application that connects MSMEs with verified local skilled workers.

## Features
- **Video Verification**: Workers upload a 30-60 second "Skill Intro" video.
- **GPS-Based Search**: Mobile-first discovery showing workers within a X km radius.
- **Admin Verification**: Direct dashboard for verify/rejecting worker profiles.

## Stack
- Frontend: Next.js (React)
- Backend: Firebase/MongoDB
- Auth: Phon-based OTP
- Storage: Cloudinary/Firebase
